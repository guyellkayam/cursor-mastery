# Project Management Plugins

> **TL;DR**: Cursor plugins for PM tools — manage tickets, track sprints, and update status from your editor.

| Plugin | Install | What It Does |
|--------|---------|-------------|
| **Linear** | `/add-plugin linear` | Issues, sprints, roadmaps, team workflows |
| **Atlassian** | `/add-plugin atlassian` | Jira issues + Confluence docs |
| **monday.com** | `/add-plugin monday` | Boards, items, status updates |

See [resources/project-management/plugins.md](../project-management/plugins.md) for detailed usage and MCP alternatives.
