# Data & Analytics Plugins

> **TL;DR**: Cursor plugins for data platforms and analytics tools.

| Plugin | Install | What It Does |
|--------|---------|-------------|
| **Databricks** | `/add-plugin databricks` | Data lakehouse, PySpark, SQL |
| **Snowflake** | `/add-plugin snowflake` | Data warehouse, SQL queries |
| **Hex** | `/add-plugin hex` | Interactive data analysis, notebooks |
| **Amplitude** | `/add-plugin amplitude` | Product analytics, user behavior |
| **Datadog** | `/add-plugin datadog` | Monitoring, APM, log analysis |
| **Grafana Labs** | `/add-plugin grafana` | Metrics visualization, dashboards |
| **Braintrust** | `/add-plugin braintrust` | AI model evaluation and testing |
| **Firecrawl** | `/add-plugin firecrawl` | Web scraping, data extraction |
| **Elastic** | `/add-plugin elastic` | Search and analytics |
| **Tavily** | `/add-plugin tavily` | Web search and research |
| **Azure Cosmos DB** | `/add-plugin cosmos-db` | NoSQL database |

## "Use this when..."

| Scenario | Plugin |
|----------|--------|
| Big data / lakehouse queries | Databricks |
| SQL analytics and reporting | Snowflake |
| Interactive data exploration | Hex |
| Product analytics and funnels | Amplitude |
| Application monitoring and debugging | Datadog |
