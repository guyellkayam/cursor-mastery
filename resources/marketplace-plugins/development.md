# Development Plugins

> **TL;DR**: Cursor plugins for development platforms — source control, deployment, and backend services.

| Plugin | Install | What It Does |
|--------|---------|-------------|
| **GitHub** | `/add-plugin github` | PRs, issues, code review, Actions |
| **GitLab** | `/add-plugin gitlab` | MRs, pipelines, issues |
| **Vercel** | `/add-plugin vercel` | Deploy, serverless functions, edge config |
| **Netlify** | `/add-plugin netlify` | Deploy, functions, forms, edge |
| **Cloudflare** | `/add-plugin cloudflare` | Workers, D1, R2, Pages, KV |
| **Supabase** | `/add-plugin supabase` | Database, auth, storage, edge functions |
| **PlanetScale** | `/add-plugin planetscale` | MySQL, branching, deploy requests |
| **Glean** | `/add-plugin glean` | Enterprise search, knowledge base |
| **Render** | `/add-plugin render` | Cloud infrastructure, deployment |
| **Redis** | `/add-plugin redis` | In-memory data store integration |
| **Clerk** | `/add-plugin clerk` | Authentication, user management |
| **JFrog** | `/add-plugin jfrog` | DevOps, artifact management |
| **Postman** | `/add-plugin postman` | API lifecycle management |
| **Elastic** | `/add-plugin elastic` | Search and analytics engine |
| **Svelte** | `/add-plugin svelte` | Svelte/SvelteKit development tools |

## "Use this when..."

| Scenario | Plugin |
|----------|--------|
| Managing PRs and code review | GitHub / GitLab |
| Deploying frontend/fullstack apps | Vercel / Netlify |
| Edge computing, CDN, Workers | Cloudflare |
| Full-stack with managed database | Supabase |
| MySQL with branching workflow | PlanetScale |
| Enterprise knowledge search | Glean |
