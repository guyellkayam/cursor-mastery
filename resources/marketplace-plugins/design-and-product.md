# Design & Product Plugins

> **TL;DR**: Cursor plugins for design tools and CMS platforms.

| Plugin | Install | What It Does |
|--------|---------|-------------|
| **Figma** | `/add-plugin figma` | Design-to-code, component inspection, asset export |
| **Webflow** | `/add-plugin webflow` | CMS management, site auditing, publishing |
| **Sanity** | `/add-plugin sanity` | Structured content CMS, GROQ queries |
| **Box** | `/add-plugin box` | Content search, document management, AI Q&A |
| **Contentful** | `/add-plugin contentful` | Headless CMS, content modeling, entry management |
| **Miro** | `/add-plugin miro` | Board access — AI reads context, creates diagrams, generates code; enterprise-grade security |
| **Meta Reality Labs** | `/add-plugin meta-reality-labs` | Quest/Horizon OS dev — debugging, perf tracing, code review, porting; Unity, Spatial SDK, WebXR; uses hzdb CLI |

## "Use this when..."

| Scenario | Plugin |
|----------|--------|
| Implementing designs from Figma | Figma |
| Managing Webflow CMS content | Webflow |
| Working with structured content | Sanity / Contentful |
| Document management and search | Box |
