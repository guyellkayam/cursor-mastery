# Community Plugins

> **TL;DR**: Community-built plugins available via cursor.directory — the open hub for Cursor extensions.

## cursor.directory

**URL**: [cursor.directory](https://cursor.directory)
**Community**: 75,000+ developers

### What's Available
- **Plugins**: Community-built bundles of MCP + Skills + Rules
- **MCP Servers**: 75+ external tool integrations
- **Rules**: .mdc files for various frameworks and domains
- **Events**: Community meetups and workshops
- **Jobs**: AI engineering positions

### How to Browse
1. Go to [cursor.directory](https://cursor.directory)
2. Filter by category (plugins, MCP servers, rules)
3. Click to view details and install instructions
4. One-click install for supported plugins

### How to Submit Your Own
1. Go to [cursor.directory/plugins/new](https://cursor.directory/plugins/new)
2. Sign in with GitHub or Google
3. Components are auto-detected if following Open Plugins standard
4. Rules auto-detected at path `rules/*.mdc`
